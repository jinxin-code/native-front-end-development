# Native Front-end Development

## 中文版本

### 仓库简介

本仓库用于存储原生前端开发的代码示例和项目，展示如何使用纯 HTML、CSS 和 JavaScript 实现各种功能，无需依赖任何前端框架。

### 项目列表

#### 1. 用户管理系统 (`user-management/`)
- **功能**：用户卡片展示、搜索、筛选、编辑、新增、删除
- **技术**：纯 JavaScript + HTML + CSS
- **特色**：单页应用页面切换效果、防抖搜索、XSS 防护

### 技术理念

1. **无框架依赖**：仅使用浏览器原生 API
2. **轻量级**：无需构建工具，直接运行
3. **学习友好**：代码结构清晰，易于理解
4. **性能优先**：避免框架带来的额外开销

### 目录结构

```
├── user-management/      # 用户管理系统
│   ├── index.html        # 主页面
│   └── README.md         # 项目说明
└── README.md             # 仓库说明
```

### 使用方法

每个项目都是独立的，直接用浏览器打开项目目录下的 `index.html` 文件即可运行。

## English Version

### Repository Introduction

This repository stores code examples and projects for native front-end development, demonstrating how to implement various features using pure HTML, CSS, and JavaScript without any front-end frameworks.

### Project List

#### 1. User Management System (`user-management/`)
- **Features**: User card display, search, filter, edit, create, and delete
- **Technology**: Pure JavaScript + HTML + CSS
- **Highlights**: SPA page switching effect, debounced search, XSS protection

### Technical Philosophy

1. **Framework-free**: Only uses native browser APIs
2. **Lightweight**: No build tools required, runs directly
3. **Learning-friendly**: Clear code structure, easy to understand
4. **Performance-first**: Avoids framework overhead

### Directory Structure

```
├── user-management/      # User Management System
│   ├── index.html        # Main page
│   └── README.md         # Project documentation
└── README.md             # Repository documentation
```

### Usage

Each project is standalone. Simply open the `index.html` file in your browser to run.
